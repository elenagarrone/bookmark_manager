Bookmark manager
================

During week four we built a bookmark manager, similar to pineapple.io or delicious.com in spirit. A bookmark manager is a good use case for exploring how relational databases work.

Technologies:
------------
- Ruby
- Rspec
- Cucumber
- Capybara
- Sinatra
- PSQL and Postgres
- BCrypt
- DataMapper
- HTML/CSS
